# Starter Template: A One-Page Personal Site

This folder holds an optional starting point for your personal website: one HTML file and one CSS file.
It exists so you do not spend your limited Copilot requests on boilerplate, which matters a lot if you are on Copilot Free.
Dropping it in and filling in the placeholders takes about 10 minutes.

## What is in here

- [`index.html`](index.html): the page itself, with a hero, an about section, three project cards, and a footer.
- [`style.css`](style.css): the styles. The variables at the top of the file are the fastest thing to change.

You should already have a live hello-world page from [03_publish_your_first_page.md](../guides/03_publish_your_first_page.md).
This template replaces that hello-world page with something worth showing off.

## How to use it

The step-by-step walkthrough for copying these files into your site lives in the "Start from the starter" section of [04_vibe_code_your_website.md](../guides/04_vibe_code_your_website.md).
Follow the numbered steps there.
The short version: copy `index.html` and `style.css` into your `username.github.io` folder, fill in the placeholders marked with comments like `<!-- your name goes here -->`, preview, then commit and push.

> ⚠️ **Important:** this site is public to the whole internet.
> Do not put your home address, phone number, or other sensitive details on it.
> The avatar is a plain circle with your initials on purpose: only add a real photo if a parent or guardian is okay with it.

## Preview before you push

You installed Microsoft's **Live Preview** extension back in [02_setup_vscode_and_copilot.md](../guides/02_setup_vscode_and_copilot.md).
If you skipped that step, the install instructions are in that guide.

1. Right-click `index.html` in the file list and choose **Show Preview**.
   You should now see your page in a tab inside VS Code, and it refreshes every time you save.

No extension handy? Double-click `index.html` in File Explorer or Finder to open it in your browser, then refresh the tab after each save.

## It is a launchpad, not a cage

Using this template is completely optional.
Keep all of it, keep half of it, or delete it and have Copilot build something wilder from scratch.
The plain look is on purpose: this is a wireframe waiting for your art direction, not a design you are stuck with.
One restyle prompt can make it unrecognizable (see "One prompt, whole new look" in [04_vibe_code_your_website.md](../guides/04_vibe_code_your_website.md)), so no two sites that start from this file should end the day looking alike.
The only goal is a site that feels like yours.
